Password Checker

This project includes two Python scripts for checking if passwords have been compromised using the Pwned Passwords API. The first script, `checkmypass.py`, checks passwords provided as command-line arguments by hashing them and querying the API. The second script, `pwd_checker.py`, allows users to input passwords via a temporary file opened in Vim, which are then checked against the API. The second script is particularly important as it prevents passwords from being stored in command logs, enhancing security by keeping sensitive information out of the shell history. Both scripts use functions to handle API requests, process data, and provide feedback on password safety. The use of SHA-1 hashing and API integration ensures an effective and secure method for password checking.
